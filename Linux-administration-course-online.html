Linux Administration course is designed and taught by real-time experts who has more than 15 years of industry experience. After completion of this course, you will be able to administer or manage Linux servers and services on your own.

Linux is a popular open-source operating system. There are plenty of opportunities with Linux administration knowledge. Linux is the widely used operating system on the cloud. Linux administration is the basic course and mandatory course to build your career into DevOps.

Duration:2-3 Weeks

Prerequisites for this course:
  • A laptop or desktop with 8GB or more RAM or a basic laptop which can be used to create and practice on a remote cloud instance.
  • Basic operating system knowledge is a plus.
Curriculum:
  1. Introduction to Linux Operating System
  2. Installation and Initialization
  3. Basic commands
  4. Managing file systems and partitions
  5. Logical Volume Management(LVM)
  6. User and Group Administration
  7. Network Configuration and Troubleshooting
  8. Security
  9. Booting procedure and kernel parameters
  10. Job automation
  11. Remote Systems administration
  12. Package management
  13. File Sharing (FTP,NFS and Samba)
  14. DNS(Domain Name Server)
  15. Web Servers( Apache,Nginix and tomcat)
  16. Different installation methods(Kickstart &Network)
  17. Mock Interview and Interview Questions
